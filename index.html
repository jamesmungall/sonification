Solar Sounds A sonification of Open Data provided by NASA and other agencies This project has developed a range of tools to help people sonify open astrophyical data.
  • Tool to Download NASA solar data images as png
  • Download Scratch Code which can produce sounds that give a flavour of activity in the various frequecy ranges of solar images
  • Processing individual images into a histogram formatted as a csv.
  • Tool to sonify the csv histogram data
  • Unity 3D Model Solar System
